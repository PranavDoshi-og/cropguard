CropGuard | Innovate You Techathon 3.0
CropGuard is an advanced, AI-driven agricultural ecosystem developed for Innovate You Techathon 3.0 (2026). The platform provides a centralized hub for farmers to diagnose crop diseases, monitor regional health trends, and access expert agricultural guidance through a high-performance, accessible interface.

🌟 Project Highlights
Intelligent Diagnostics: Leverages a Random Forest Classifier to analyze plant pathology from leaf imagery, providing instant feedback on crop health.

Centralized Farm Management: Implements a robust data management system (inspired by AlumHub) to track history, growth cycles, and treatment efficacy across multiple plots.

Localized Accessibility: Designed for rural adoption with a multi-lingual interface supporting English, Hindi, and Marathi, alongside voice-guided navigation for low-literacy accessibility.

Community Threat Mapping: A collaborative intelligence module that generates regional health heatmaps to alert users of nearby pathogen outbreaks.

Actionable Remediation: Provides dual-path treatment recommendations—Organic and Chemical—integrated with government subsidy data for immediate farmer support.

💻 Technical Architecture
Frontend Implementation
React.js (Vite): For a lightning-fast, component-based single-page application.

Tailwind CSS v3.4: A custom-themed "Glassmorphism" UI optimized for field visibility and dark-mode performance.

State Management: Centralized React hooks to manage cross-component data flow between the Scanner, History, and Dashboard.

Core Intelligence
Ensemble Learning: Utilizing Random Forest architecture to ensure high accuracy and low latency in diagnostic results.

Data Persistence: A structured storage system for managing scan records and user-specific farm profiles.
